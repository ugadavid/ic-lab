# Prototype 06 V1.1.3 - Lecture seule AIConfig

Date : 2026-07-02

## Objectif

Permettre a Prototype 06 de recevoir `aiConfigId` dans l'URL de lancement du hub, sans activer de vraie IA.

## Fichiers crees

- `index-1.1.3.html`
- `script-1.1.3.js`
- `style-1.1.3.css`

## Comportement

Prototype 06 lit le parametre URL `aiConfigId` et l'ajoute a l'etat de lancement hub.

En mode debug, le panneau affiche :

- `aiConfigId`;
- le fait qu'aucune IA reelle n'est activee.

La carte enseignante affiche aussi une mention informative.

## Limites

- La configuration n'est pas resolue depuis le hub.
- Aucun provider IA n'est appele.
- Aucun texte ou audio IA n'est genere.
- La V1.1.3 conserve le fonctionnement V1.1.2 pour activites, runs et evenements.

## Verification

- `node --check script-1.1.3.js`
