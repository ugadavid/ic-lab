# IC-Hub V0.8.4 - AI admin safety onboarding

Date: 2026-07-03

## Objectif

La V0.8.4 ajoute des garde-fous lisibles pour une adoption universitaire prudente. Elle ne change pas le fonctionnement IA effectif : aucune generation IA n'est activee dans Prototype 06, aucun texte apprenant n'est envoye a un provider.

## Retour arriere d'urgence

Une section visible explique comment revenir a l'etat sur :

1. verifier le runtime scenarise;
2. desactiver `allowedForRuntime` sur les modeles douteux;
3. repasser les AIConfig experimentales en `planned` ou `disabled`;
4. verifier le bandeau "aucune IA generative active";
5. relancer le hub en JSON si besoin;
6. consulter le guide mainteneur.

Aucun bouton destructif automatique n'est ajoute.

## Guide mainteneur

Le bloc `Si vous reprenez ce projet` contient un lien vers :

`docs/ai-admin-maintainer-guide.md`

Le serveur peut lire ce fichier via une route statique bornee sur `/docs/`.

## Ce qui declencherait reellement l'IA

La V0.8.4 explique que synchroniser OpenAI ne fait que lire une liste de modeles cote serveur. Pour activer une IA generative, il faudrait explicitement creer ou brancher un runtime de generation, associer une AIConfig, tester et valider.

Phrase cle ajoutee :

`Aucun de ces elements n'est actif dans cette version.`

## Donnees apprenantes

La page indique clairement :

- reponses apprenantes stockees dans les traces : non, seulement longueur / presence selon le flux existant;
- reponses apprenantes envoyees a OpenAI : non;
- textes generes par IA stockes : non;
- audio IA genere : non;
- donnees personnelles envoyees a un provider : non, dans l'etat actuel du prototype.

## Checklist avant activation future

La page ajoute une checklist de bonnes pratiques projet :

- cadre institutionnel;
- information des apprenants;
- minimisation des donnees;
- pas de texte libre apprenant sans justification;
- tests avec prompts fictifs;
- limites de duree et cout;
- logs non sensibles;
- desactivation immediate;
- validation pedagogique.

## Limites restantes

- La checklist institutionnelle reste une aide projet, pas un avis juridique.
