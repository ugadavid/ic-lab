# IC-Hub V0.9.6 - Admin IA en app-shell

Date: 2026-07-03

## Objectif

Transformer l'administration IA V0.9.5 en interface de travail avec navigation permanente et vues separees, sans ajouter de capacite IA ni changer la logique metier.

## Changements UX

- Ajout d'une app-shell admin avec topbar fixe, navigation laterale fixe et zone de travail principale.
- Une seule vue principale est visible a la fois.
- Navigation par hash URL : `#dashboard`, `#catalog`, `#aiconfig`, `#generate`, `#evaluate`, `#compare`, `#security`, `#system`.
- Les vues Generer, Evaluer et Comparer reutilisent le meme AI Lab sans dupliquer la logique metier.
- Les anciennes versions restent disponibles par leurs fichiers versionnes.

## Workflow admin

- Generer : choix modele/scenario/personnage, generation admin-only et export draft pack.
- Evaluer : import local de draft pack, grille locale, commentaires, favoris et export evaluation.
- Comparer : import local de plusieurs exports evaluation, agregats et export comparison.
- Systeme : health, stockage, utilisateurs, cours, prototypes, connecteurs et sessions.

## Securite conservee

- Aucun appel OpenAI depuis le navigateur.
- Aucune cle API affichee.
- Aucun contenu `.env` affiche ou logge.
- Aucun texte apprenant envoye.
- Aucun stockage serveur des brouillons, evaluations, imports ou comparaisons.
- Aucun ecriture dans `run_events`.
- Aucun changement Prototype 06.
- JSON et MariaDB restent conserves.
