Administration des IA

              
Cette page sert a gerer les fournisseurs IA, les modeles disponibles et les modeles autorises pour IC-Lab. Elle ne lance aucune IA dans les activites apprenantes.

            

            Aucune generation activee
          


          

            

              
Provider 

              
Un provider est un fournisseur technique : OpenAI, Anthropic, Google, Mistral, Ollama, etc.

            

            

              
Modele 

              
Un modele est un moteur utilisable chez un provider : modele texte, audio, image, embedding, temps reel, etc.

            

            

              
Synchronisation 

              
La synchronisation OpenAI demande au serveur local quels modeles sont accessibles avec la cle configuree. La cle reste cote serveur et n'est jamais affichee.

            

          


          

            

              
Catalogue OpenAI

              
Met a jour la liste locale des modeles accessibles. Cette action n'envoie aucun texte apprenant a OpenAI.

Comprendre les couts

            
Les couts affiches ici sont indicatifs. La synchronisation OpenAI ne fournit pas automatiquement les prix detailles. Les prix doivent etre renseignes ou verifies manuellement a partir des tarifs officiels. Quand le prix n'est pas connu, le Hub affiche "Prix non renseigne".

            

              free aucun cout API
              low cout faible
              medium cout intermediaire
              high cout eleve
              very-high cout tres eleve
              unknown prix non renseigne
