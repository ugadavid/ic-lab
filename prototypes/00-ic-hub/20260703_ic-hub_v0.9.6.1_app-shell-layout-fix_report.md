# IC-Hub V0.9.6.1 - Correction layout app-shell admin IA

Date: 2026-07-03

## Objectif

Corriger la V0.9.6 sans ajouter de capacite IA : topbar non couvrante, sidebar stable, contenu bien decale et navigation unique par sidebar.

## Probleme corrige

- La topbar etait fixe mais sa hauteur reelle pouvait depasser le decalage prevu par le contenu.
- La navigation horizontale issue de la V0.9.5 restait confuse : elle ressemblait a une navigation secondaire alors que la sidebar etait deja la navigation principale.

## Correction layout

- Ajout de variables CSS simples :
  - `--topbar-height: 72px`
  - `--sidebar-width: 220px`
- Topbar fixee a une hauteur explicite.
- Sidebar fixee sous la topbar, avec hauteur `calc(100vh - var(--topbar-height))`.
- Zone centrale decalee avec `margin-top: var(--topbar-height)` et `margin-left: var(--sidebar-width)`.
- Largeur de vue portee a `1280px` maximum pour eviter une interface trop tassee.
- Responsive minimal ajoute pour ecrans etroits.

## Correction navigation

- Suppression de la navigation horizontale `workspace-nav` dans `admin-0.9.6.html`.
- Suppression des styles CSS morts associes.
- La sidebar reste la navigation de reference.
- Les clics sidebar mettent a jour le hash via `history.pushState`.
- Le retour navigateur est pris en charge via `popstate`.
- Le rechargement sur un hash, par exemple `#compare`, rouvre la bonne vue.
