# IC-Hub V0.9.4.1 - Correctif import draft pack

Date: 2026-07-03

## Probleme

Dans `admin-0.9.4`, l'import d'un draft pack valide pouvait echouer avec :

`countWords is not defined`

Le chemin d'import draft pack appelait `countWords(...)`, mais ce helper n'etait pas defini dans `public/admin-0.9.4.js`.

## Correction

Fichier modifie :

- `public/admin-0.9.4.js`

Ajouts :

- fonction locale sure `countWords(text)`;
- import draft pack plus robuste;
- utilisation de `wordCount` existant si fourni;
- fallback vers `countWords(text)` si `wordCount` manque;
- erreur lisible si un brouillon importe ne contient aucun texte.

## Comportement attendu

Un draft pack valide comme `lot_2.json` peut etre importe sans relancer OpenAI. Les brouillons importes deviennent des brouillons de session evaluables localement avec grille, commentaire et favori.

## Securite

Le correctif ne change aucun endpoint serveur :

- aucun upload serveur;
- aucun appel OpenAI navigateur;
- aucun stockage DB;
- aucun `run_event`;
- aucun `localStorage` pour brouillons ou evaluations;
- Prototype 06 non modifie.

## Verification realisee

- `node --check public/admin-0.9.4.js` OK.
- Presence de `countWords`.
- Presence du fallback `wordCount` existant puis comptage local.
- Presence du message `brouillon X sans texte`.

Les tests interactifs d'import de `lot_2.json` sont a faire dans le navigateur avec le fichier local de l'utilisateur.
